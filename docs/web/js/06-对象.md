# 循环语句
英文名称：`Object`。
类似于`C++`中的`map`，由`key:value`对构成。

- value 可以是变量、数组、对象、函数等。
- 函数定义中的`this`用来引用该函数的“拥有者”。


例如：

```javascript
let person = {
    name: "yxc",
    age: 18,
    money: 0,
    add_money: function (x) {
        this.money += x;
    }
}
```
对象属性与函数的调用方式：
- `person.name`、`person.add_money()`
- `person["name"]`、`person["add_money"]()`
